# PRD — 사주 웹 서비스 (MVP)

## 목표
Y Combinator 배치를 위한 사주 웹앱 초고속 출시 + 가설 검증.
**"속도가 생명"** — 바퀴를 재발명하지 않되, 트래픽 폭발에도 무너지지 않는 안정성 확보.

---

## 핵심 가설
1. MZ 세대는 "데이터화된 사주" (도넛 차트 + 성격 카드 + 카테고리 운세)를 재미있어한다.
2. 공유 경험이 바이럴 루프를 만든다. 결과 화면 자체가 SNS 명함이 되어야 한다.
3. 카카오 OAuth 단일 인증 + 단순 입력폼이 가입 허들을 최소화한다.

---

## 사용자

**P1 (메인)** — 사주 처음 접하는 20-30대 MZ
- 진입 경로: 친구 공유 링크 or 검색
- 욕구: 재미있는 자기 탐색 + 친구에게 공유
- 비선호: 명리학 용어 폭격, 긴 가입 폼, 무미건조한 결과

---

## 기능 범위 (MVP)

### In Scope
| 기능 | 설명 |
|---|---|
| 카카오 OAuth 로그인 | 단일 인증, 가입폼 없음 |
| 사주 차트 생성 | 양력/음력, 생년월일(시), 성별 입력 |
| 결과 화면 | 사주 4기둥 + 5행 도넛 + 성격 카드 6장 |
| 카테고리 운세 6종 | 총운/금전/연애/건강/직업학업/가족인간관계 |
| 오늘의 운세 | SELF 차트 전용, 일진 기반 |
| 지인 차트 | subject_kind=OTHER, 이름 입력 |
| 공유 | 8자 slug URL, 1년 만료, PII 마스킹 |
| 이력 목록 | 내가 만든 차트 목록 (offset 페이지) |
| 회원 탈퇴 | 30일 유예 소프트 삭제 (카카오 정책) |

### Out of Scope (Post-MVP)
- 커서 페이지네이션
- 푸시 알림
- 앱스토어 배포
- 사주 해설 직접 입력 (LLM 프리컴퓨트로 대체)

---

## UX 잠금 원칙 (변경 금지)

1. **Auth Wall**: 로그인 전 결과 불가. 단, 공유 링크(`/s/{token}`)는 비로그인 허용.
2. **입력 차단 조건**: 미성년자 (만 14세 미만), 잘못된 날짜 — 엄격 차단.
3. **자시 모달 강제**: 00:00-01:00 입력 시 반드시 야자시/자시 선택 확인.
4. **시간 불명 lenient**: 시간 모를 경우 허용, 결과에서 "시주 미포함" 안내.
5. **공유 퍼스트**: 결과 first fold에 공유 버튼 고정.
6. **명리 용어 격리**: 용어는 "상세보기" 토글 안에만 노출.
7. **PWA 전용**: 앱스토어 없음, 홈 화면 추가만.

## 비기능 요건

| 항목 | 목표 |
|---|---|
| API P95 응답 | < 500ms (LLM 경로 없음) |
| 가용성 | 99.5% (ECS Fargate) |
| 세션 TTL | 14일 |
| 인증 방식 | Kakao OAuth → JDBC Session (Cookie) |
| LLM 콘텐츠 | 오픈 전 사전 생성, 사용자 경로에 실시간 LLM 호출 없음 |
